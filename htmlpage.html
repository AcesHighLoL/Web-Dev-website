What is HTML?

To understand why HTML is important, we must first dive into the meaning of HTML. HTML stands for Hyper Text Markup Language. It is the standard Markup Language for creating a webpage. HTML is the skeleton of a webpage. It consists of of many elements that provides the structure of a website. When creating a webpage, you want to differentiate different titles and headings. You want the computer to know "this is a header so it must stand out a these are the paragraphs".

How Is Understanding HTML helpful?

Learning HTML is helpful because it allows web developers to create codes that becomes pararagraphs, sections, and links on a webpage. HTML is responsible for how elements are displayed on webpages It is the reason for why clickable links are able to bring you to another landing page. HTML makes it possible to layout a webpage similar to that of a Microsoft Word document.

Some Simple HTML Terminology

HTML elements are represented by < > tags. Here are some quick and easy element in HTML:

  • All HYML documents begin with the document declaration: < !DOCTYPE html >
  • < html > is the root element in a HTML page
  • < title > specifies the title of that document
  • < body > contains the page content including but not limited to paragraphs and subheadings
  • < h1 > is the largest heading (if you want to create a subheading, you can use < h2 > to make the title smaller than < h1 > )
  • < p > defines every paragraph
